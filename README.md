# Steps to Deploy Frontend
## Prerequisites
- [Node.js](https://nodejs.org/) (version 14.x or later)
- [Angular CLI](https://angular.io/cli) (version 20.3.9)
- A package manager like [npm](https://www.npmjs.com/) (comes with Node.js) or [Yarn](https://yarnpkg.com/)
## Environment Setup
- Replace `ASP_API_URL` and `DJANGO_API_URL` in `src/environments/environment.ts` with the appropriate backend API URLs.
## Run
1. `npm install`
2. `npm run start` or `ng serve`
